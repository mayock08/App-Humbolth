## Credenciales de Acceso (Entorno de Pruebas/Desarrollo)

A continuación se detallan las credenciales por defecto para los diferentes roles de usuario.

### Administrador
- **Usuario:** `admin@edu.com`
- **Contraseña:** `password`

### Profesor (Teacher)
- **Usuario:** Correo electrónico del profesor (ej. `teacher@edu.com`) o Matrícula
- **Contraseña:** `teacher123` (Por defecto si no se ha establecido una personalizada)

### Coordinador
- **Usuario:** Correo electrónico del coordinador
- **Contraseña:** `coordinator123` (Por defecto)

### Estudiante
- **Usuario:** Correo electrónico o Matrícula del estudiante
- **Contraseña:** `student123` O la misma Matrícula (para primer acceso)

### Padre de Familia (Guardian)
- **Usuario:** Correo electrónico o Teléfono móvil
- **Contraseña:** `parent123`
